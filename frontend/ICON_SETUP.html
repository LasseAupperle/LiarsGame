🎨 Liars Game - PWA Icon Setup

⚠️ Placeholder Icons
The app currently uses placeholder icons. Replace them with your game artwork.

Quick Setup (Online)

  1. Visit: PWA Builder Image Generator
  2. Upload your logo/icon (512x512 or larger)
  3. Download the generated icons
  4. Extract to frontend/public/icons/

Manual Setup (Command Line)

If you have ImageMagick installed:

cd frontend
chmod +x generate-icons.sh
./generate-icons.sh path/to/your-logo.png

Required Files

  • public/icons/icon-192.png
  • public/icons/icon-512.png
  • public/icons/icon-192-maskable.png
  • public/icons/icon-512-maskable.png
